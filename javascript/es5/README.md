### # Variable naming

**Always using lowerCamelCase to naming your variable.**

Bad

```
var MYCARD = 'red car'

var my-posts = ['Hello webdoc']
```

Good

```
var myCard = 'red car'

var myPosts = ['Hello webdoc']
```
